HTML Table Reference


HTML Tables

Tag Name Description
<table> Table The container for the entire table
<thead> Table Head The table header section
<tbody> Table Body The section containing the table data
<tfoot> Table Footer The footer section of the table
<tr> Table Row A table row
<th> Table Header Cell A header cell (bold and centered)
<td> Table Data Cell A data cell
HTML TABLE TAGS

Table Attributes


Attribute Name Description
<colspan> Column Span Defines how many columns a cell spans
<rowspan> Row Span Defines how many rows a cell spans
